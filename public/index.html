About this page

This site contains a home automation system, which I use to experiment with different technologies. It is written by myself for myself.

Information about this project can be read in the Readme.
You can also check it out on Github
The code is messy and unorganized, as it has grown "organically". The system assumes the presence of components such as an Arduino, a Mopidy instance, and a MySQL server, and thus likely won't run on other systems.
It might however provide some inspiration for other developers.

For feedback, contact me at tobiashomeautomation@gmail.com.

The main interface for the home automation can be viewed at Music
